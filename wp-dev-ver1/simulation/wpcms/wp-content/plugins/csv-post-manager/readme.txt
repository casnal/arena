== Description ==

The CSV Post Manager plugin adds the functionality to manage posts by using csv files.

== Installation ==

1. Copy the `csv-post-manager` directory into your `wp-content/plugins` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. That's it! :)
